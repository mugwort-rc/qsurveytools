[![License: GPL v3](https://img.shields.io/badge/License-GPL%20v3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

アンケート等の集計ツールです。

## Linux

### Install Dependencies

```
pip install -r requirements.txt
```

### Make PyQt files

```
./tools/init-qt-resource.sh
```

### Usage

```
python main.py
```
