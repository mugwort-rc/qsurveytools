# 集計

![mainwindow.png](img/mainwindow.png)

## 操作法

1. **入力**に[入力ファイル]を設定します。
2. **出力**に出力先のフォルダを設定します。

**Tips:**

* 右の`...`ボタンを押すと選択ダイアログが起動します。
* 入力エリアにファイルをドラッグ＆ドロップすると自動で入力されます。

### 無回答を除く

**無回答を除く**にチェックを入れると、集計から無回答を除きます。

**単純集計**

全体は無回答を除いた数が出力されます。

**クロス集計**

表頭・表側の一方が無回答である回答を除いた数が出力されます。
